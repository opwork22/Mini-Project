# Student Data Preprocessing & Visualization Project

## Project Overview
This project demonstrates **data preprocessing, validation, cleaning, analysis, and visualization**
on a student dataset using **Python**, **Pandas**, **CSV**, and **Plotly**.

The project handles:
- Missing values
- Invalid marks
- Duplicate roll numbers
- Manual user correction
- Result & grade calculation
- Interactive data visualization

This project is useful for **learning data preprocessing fundamentals** before applying Machine Learning.
